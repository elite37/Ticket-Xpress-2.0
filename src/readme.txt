// rules for css modules 

-- convert css to module css by changing the extension name
 e.g style.css >> style.module.css

-- you cannot have className with hyphen
e.g footer-container 
instead use footer__container or footerContainer or footer_container 

-- you cannot use id , only className is valid
  so change all id instance to className and make the necessary changes in the css
